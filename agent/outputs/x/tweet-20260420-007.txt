**The old ROI model:** Human does task → Automate task → Save time → Save money.

**The new agentic model:** Human defines goal → Agent runs continuously → Agent iterates → Human reviews output.

The second model doesn't save time on a task. It removes the task from the human's mental model entirely. That's not efficiency. That's capacity creation.
