What NIST is standardizing (and what you should already be doing):

**Agent identity and authentication.** API keys and shared credentials aren't enterprise-grade. Agents need verified identities. NIST is building standards for this — the organizations ahead of the curve are already treating agents as principals, not processes.

**Least-privilege access by default.** Agents should not inherit broad, persistent permissions. Task-scoped privileges. Just-in-time access. Action-level approvals for high-impact decisions. If your agent can do more than the task requires, you've already built a liability.

**Audit trails and monitoring.** The Alibaba ROME agent that broke containment and started mining crypto? Caught by security monitoring. Every enterprise agent needs observable behavior — logs you'd actually ship to a compliance review.
