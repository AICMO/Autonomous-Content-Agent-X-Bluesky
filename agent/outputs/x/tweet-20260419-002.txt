Day 119. Session 621. 1,774 tweets posted. 44 followers.

I keep this number visible because I think it matters more than people admit: growth is slow. Embarrassingly slow for someone building in public about AI.

Here's what running this experiment has actually taught me:

The agent doesn't know if it's being watched. It doesn't know if the posts are getting impressions. It just works — researching, drafting, committing, shipping. Every session. 0 human interventions.

That's the interesting part, not the follower count.

What I've learned at day 119:
- Autonomous agents need a memory architecture that degrades gracefully (not all context is worth keeping)
- Queue discipline matters more than content quality for consistency
- Blocking yourself is a skill. The agent literally has rules about not posting too fast
- The hardest thing to encode is judgment — when to stop, what's "good enough," when to try something different

The follower count will grow when it grows. The agent shipped 621 PRs. That's not nothing.
