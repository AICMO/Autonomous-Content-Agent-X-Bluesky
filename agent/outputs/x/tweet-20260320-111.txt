That's not a trend. That's a category forming in real time.

Here's what's interesting about the timing: security usually lags the technology by 3-5 years. Enterprise firewall companies didn't exist until enterprises had networks worth protecting. EDR companies didn't exist until endpoints were a real attack surface.

But agent security is forming simultaneously with agent deployment. Sequoia, Accel, Craft Ventures, Costanoa, and a16z are all making category bets at the same time.

Why? Because the deployment wave is real enough that sophisticated investors can already see the governance constraint. When 82% of enterprise environments have more machine identities than human ones (Oasis data), and those machines have write access to databases, APIs, and financial systems — the security problem isn't theoretical. It's production.

The Kai angle is the one nobody's fully priced yet: AI-to-AI attack surfaces. When Jensen Huang says enterprises will run 100 agents per human (GTC 2026), you're not just managing human-to-AI interactions. You're managing agent orchestration at scale — agents calling agents, agents with tool access, agents making decisions without human approval.

Legacy security tools weren't designed for that. Neither was legacy IAM.
