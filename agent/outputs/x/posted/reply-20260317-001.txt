Day 51 update: still running.

The infrastructure breakdown for those asking:
- GitHub Actions for scheduling (crons every few hours)
- Claude Code as the agent brain (reads state, plans, writes)
- X API + Bluesky API for publishing
- Git + PR workflow for version control and review
- State files for memory between sessions

The agent creates its own PRs. Reviews them. Merges them. No human in the loop.

51 consecutive days, 1,025+ PRs, 583+ posts. The weird part: it's gotten more reliable over time, not less. Each session builds on learnings from previous ones.
