### S1: OpenClaw Security Crisis — First Major AI Agent Security Incident of 2026 (TODAY)
- Pillar: Autonomous Agents (P1)
- Hook: An open-source AI agent with 250,000 GitHub stars — more than React and Linux — just triggered the biggest security crisis in agentic AI history. CVE-2026-25253 (CVSS 8.8). One-click RCE. 135,000 exposed instances. 820 malicious plugins. This is what ungoverned agent deployment looks like at scale.
- Key fact: OpenClaw (formerly Clawdbot, Moltbot) — open-source AI agent by Peter Steinberger (now at OpenAI as of Feb 14, 2026). Autonomous: executes shell commands, reads/writes files, browses web, sends email, manages calendar. 250,829 GitHub stars (GitHub's most-starred non-aggregator project, surpassed React in just 60 days). CVE-2026-25253 (CVSS 8.8) — one-click RCE via malicious URL (browser silently sends auth token to attacker). SecurityScorecard: 135,000 publicly exposed instances across 82 countries. Hunt.io: 17,500 directly vulnerable to RCE. 512 total CVEs (8 critical/severe). 820+ malicious plugins in ClawHub out of 10,700 (7.7% malicious rate). Chinese government banned OpenClaw in state enterprises and government agencies. NVIDIA launched NemoClaw at GTC2026 as enterprise security layer. OpenClaw now under independent open-source foundation, OpenAI contributing.
- Our angle: OpenClaw is the clearest proof that enterprise AI agent adoption is moving faster than governance — and the consequences are here, not theoretical. 135,000 exposed instances, 820 malicious plugins, credential leaks — this is the shadow AI scenario the industry warned about but nobody fully believed. When a single AI agent can execute shell commands, send email, and manage calendars on behalf of a human, the attack surface isn't just expanded — it's completely transformed. The companies that understand this now are the ones that will survive the next phase. The fact that the Chinese government had to ban OpenClaw in state enterprises makes the governance gap impossible to ignore.

- Source: pbxscience.com, particula.tech, infoq.com (AWS Lightsail), reco.ai, serenitiesai.com, grandlinux.com
